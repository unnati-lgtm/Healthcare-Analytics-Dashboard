# Healthcare-Analytics-Dashboard
## Project Overview
This project analyzes healthcare data using Power BI to provide insights into patient demographics, billing, insurance providers, and medical conditions.

## Dashboard Features
- Total Patients KPI
- Average Age
- Average Billing Amount
- Medical Condition Analysis
- Gender Distribution
- Admission Type Analysis
- Insurance Provider Analysis
- Test Results Analysis
- Interactive Filters

## Tools Used
- Power BI
- Power Query
- DAX
- Excel/CSV

## Dataset
Healthcare dataset containing patient demographics, billing information, admission type, insurance provider, blood type, and medical conditions.
